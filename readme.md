# Booking of the hot air balloons

This is the repository for my hot air balloon booking project.

- Built in Golang version 1.19.2
- Uses the [chi rounter](github.com/go-chi/chi/v5) v5.0.7
- Uses alex edwards [SCS session managment](github.com/alexedwards/scs/v2) v2.5.0
- Uses [nosurf](github.com/justinas/nosurf) v1.1.1
